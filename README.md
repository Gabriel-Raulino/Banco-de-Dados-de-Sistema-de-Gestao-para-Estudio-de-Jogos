# Banco-de-Dados-de-Sistema-de-Gestao-para-Estudio-de-Jogos
anco de Dados relacional para gestão de estúdios de jogos. Gerencia RH, ciclo de desenvolvimento, vendas multiplataforma e engajamento da comunidade. O sistema centraliza dados críticos de jogos e desenvolvedores, integrando IA Generativa para análise de feedbacks e auxílio na tomada de decisão estratégica.
